# 运算符

#### 算数运算符

```python
print(1 + 1) # 2
print(4 - 2) # 2
print(3 * 2) # 6
print(9 / 2) # 4.5 在python2中为整数4 python3中为float4.5

print(3 ** 3) # 27
print(81 ** 1 / 2) # 40.5 优先级 print(81 ** (1 / 2))

print(10 / 3)  # 3.3333333333333333333
print(10 // 3) # 3 整除，只取整数部分

print(10 % 3)  # 1
```

#### 算数运算符在字符串中的使用

```python
# 加法运算符：只能用于两个字符串类型的数据，用来拼接两个字符串
print("hello" + "world") # helloworld
print('hello' * 2)       # hellohello
```

#### 赋值运算符（=）

```python
# 计算机编程中，等号作用是将等号右边的值赋值给等号的左边
# 等号的左边一定不能是常量或者表达式
+=
-=
*=
/=
**= 
//=

a = b = c = d = 10
print(a,b,c,d)      # 可以

x = 'hello','good','yes'
print(x)

y, z = 1, 2, 3, 4, 5
print(y,z) # 报错  *z就可以
```

#### 比较运算符

```python
# 大于 >  小于 <  等于 =   大于等于 >=     小于等于 <=    不等于 ！=   等等与 ==
print('a' > 90) # 报错  str和int比较报错 
# 不会用Ascall码去比较，字符串之间可以比较；数字和字符串之间，做 == 运算结果是Flase，做 ！= 结果是 True，其他的比较运算符不支持
```

#### 逻辑运算符的使用

```python 
#逻辑与 and   逻辑或 or    逻辑非not

4 > 3 and print("hello world")
4 < 3 and print("您好，世界！")

打印第一个

4 > 3 or print("嘿嘿嘿")
4 < 3 or print("嘤嘤嘤")

打印第二个

print('good' and 'yes' and 'ok' and '100')

打印100 # 打印最后一个，会运行到最后，所以打印最后一个

print（0 or [] or {} or ()）

打印0  # 打印第一个Ture的，不会再运行
```
